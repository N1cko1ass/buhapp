## Деплой на Vercel — пошаговая инструкция

### Шаг 1: Загрузите код на GitHub
1. Зайдите на github.com → New repository → назовите `buh-app`
2. Загрузите все файлы этой папки

### Шаг 2: Подключите Vercel
1. Зайдите на vercel.com → Add New Project
2. Выберите ваш репозиторий `buh-app`
3. Нажмите Deploy (ничего не меняйте в настройках)

### Шаг 3: Добавьте Vercel KV (общая база данных)
1. В панели Vercel → ваш проект → Storage
2. Нажмите Create Database → выберите KV (Redis)
3. Назовите `buh-data` → Create & Connect
4. Это автоматически добавит нужные переменные окружения

### Шаг 4: Добавьте Gemini API ключ
1. Получите ключ на: https://aistudio.google.com → Get API key
2. В Vercel → Settings → Environment Variables
3. Добавьте:
   - Name: `GEMINI_API_KEY`
   - Value: ваш ключ (начинается с AIzaSy...)
   - Environment: Production, Preview, Development
4. Нажмите Save

### Шаг 5: Передеплойте
1. Vercel → Deployments → три точки → Redeploy

## Частые вопросы

**Данные потеряются при переезде?**  
Нет — данные хранятся в Vercel KV (Redis), не в коде.

**Как сбросить данные к начальным?**  
Vercel → Storage → ваша KV база → Browser → удалите ключ `buh_shared_data`.

**Хочу чтобы у каждого были свои данные?**  
Это потребует добавления авторизации — отдельная доработка.
